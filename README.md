## Projects

### 1. Expense Tracker
**Folder:** `expense-tracker/`

**Features:**
- Add an expense with name and amount
- Display expense list dynamically
- Automatically update total expenses
- Delete individual expenses with ❌(CLOSED BUTTON) button
- Highlight expenses above 10,000 in red
- Clear all expenses with a button

---

### 2. Student Result Grade UI
**Folder:** `grade-ui/`

**Features:**
- Enter student name and score (0–100)
- Calculate grade (A–F)
- Color code grades:
  - A → Green
  - B → Light green
  - C → Yellow
  - D → Orange
  - F → Red
- Display all tested students in a list
- Clear inputs and result with a button

---

### 3. Color Theme Switcher
**Folder:** `theme-switcher/`

**Features:**
- Switch between Light, Dark, and Blue themes
- Smooth transition between themes
- Active theme button highlighted
